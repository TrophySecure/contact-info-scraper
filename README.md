# Contact Info Scraper

Extract emails, phone numbers, and social media profiles from any website. Crawls homepages and contact pages for comprehensive lead data.

## Features

- **Structured JSON output** ready for databases, spreadsheets, and APIs
- **Automatic pagination** handles multi-page results seamlessly
- **Built-in retry logic** with error handling for reliable extraction
- **Configurable inputs** to target exactly the data you need
- **Pay per result** with no monthly fees or subscriptions

## How to use

1. Click **Try for free** to open the Actor in the Apify Console
